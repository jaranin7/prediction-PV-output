Temporal Fusion Transformer

What?

ระบบนี้เป็นระบบพยากรณ์การผลิตกำลังไฟฟ้าจากแผงโซลาร์เซลล์ โดยใช้ข้อมูลจากเซนเซอร์เพื่อช่วยในการคาดการณ์การผลิตพลังงานไฟฟ้าจากพลังงานแสงอาทิตย์

read more

Where?

สถาที่ที่ใช้ในการเก็บข้อมูลคืออาคารปฏิบัติการอิเล็กทรอนิกส์ โดยมีเซ็นเซอร์ที่เก็บข้อมูลดังต่อไปนี้

1.เซ็นเซอร์แสง(Max44009)

2.เซ็นเซอร์วัดอุณหภูมิ(AM2315)

3.เซ็นเซอร์แรงดันไฟฟ้าและกระแส(PZEM-017)

read more

When?

ระบบจะเก็บข้อมูลจากเซนเซอร์ทุกๆ 5 วินาที และจัดเก็บในฐานข้อมูล MySQL ข้อมูลที่เก็บสะสม 2 วันจะถูกใช้เป็น encoder สำหรับการพยากรณ์การผลิตกำลังไฟฟ้าหนึ่งวัน โดยใช้ข้อมูลย้อนหลัง 1 เดือนในการฝึกโมเดลเพื่อเพิ่มความแม่นยำ

read more

Which?

โปรเจกต์ใช้เซนเซอร์ DHT22, Max44009, และ PZEM-017 เพื่อเก็บข้อมูลอุณหภูมิ, ความชื้น, รังสีแสงแดด, และพลังงานจากแผงโซลาร์เซลล์ โดยใช้ ESP32 ส่งข้อมูลไปยัง MySQL ผ่าน Node-RED และ PHP สำหรับการประมวลผลและแสดงผลข้อมูลบนเว็บด้วย JavaScript โดยใช้ข้อมูล 2 วันเป็น encoder ในการพยากรณ์ 1 วันจากการเทรนข้อมูลใน 1 เดือน.

read more

How?

ใช้ TFT (Temporal Fusion Transformer) ในการพยากรณ์การผลิตพลังงานจากแผงโซลาร์เซลล์ โดยการเก็บข้อมูลจากเซนเซอร์ต่าง ๆ และฝึกโมเดลด้วยข้อมูลย้อนหลัง 1 เดือนเพื่อทำนายการผลิตพลังงานในอนาคต โดยอิงจากข้อมูลที่เก็บได้จากสภาพอากาศและสภาพแวดล้อมในช่วงเวลาที่ผ่านมา.
